# Web MIDI Chord Finder

Real-time display of chords you play on your MIDI instrument. Right in
your browser, thanks to Web MIDI! :)

Works on Mac Chrome. Does not work on Mac Firefox.

Doing your own MIDI stuff in Elm is not scary!
See
[lassik/elm-webmidi-ports-minimal](https://github.com/lassik/elm-webmidi-ports-minimal)
